# Fonts for Clicker Carnage UI

This folder contains all custom and futuristic fonts used in the Clicker Carnage UI/UX redesign.

- Place .ttf, .otf, or .woff files here.
- Example fonts: Orbitron, JetBrains Mono, Share Tech Mono.
- To add a new font, download the file and place it in this directory, then reference it in CSS via @font-face.
